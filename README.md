# AutoTrack - Job Tracker

**AutoTrack** is a smart job-tracking tool that helps job seekers automatically save and organize their job applications directly from platforms like **LinkedIn** and **Naukri**.

## ðŸ§© Overview

AutoTrack eliminates the need for manual spreadsheets. It automatically detects job application clicks and opens a popup to save job details such as:

* Job Title
* Company Name
* Job URL
* Application Status

All saved jobs appear neatly organized in your personal dashboard.
